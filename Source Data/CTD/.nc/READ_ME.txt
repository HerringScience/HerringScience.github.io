---------------------------
METHODOLOGICAL INFORMATION
---------------------------

1. Description of methods used for collection/generation of data: 
Temperature and salinity casts were gathered from various sources and grouped together according to year. 

2. Methods for processing the data: 
Casts were bin-averaged to a 1 (0-1000dbar),10 (1000-2000dbar),100 (2000-5000dbar) dbar vertical resolution up to a maximum of 5000 dbar. 
Multiple quality assurance/control measures were implemented in order to remove empty, duplicate, or outlier casts from the data product (reference manuscript for more detail).
All casts were saved in yearly NetCDF files with dimensions time and depth (level).

3. Instrument or software-specific information needed to interpret the data: 
All casts are saved in NetCDF file type.
Any programming language capable of handling NetCDF files, can be used.
An example python command can be used to import the entire data product with one line using the XARRAY package: "ds = xarray.open_mfdataset("*.nc")".

4. Describe any quality-assurance procedures performed on the data: 
Casts taken from Northwest Atlantic Fisheries Centre (NAFC) - Oceanography did not have any prior quality-assurance procedures.
Empty, duplicate, or outlier casts from NAFC - Oceanography were removed separately before combining with other sources (see manuscript).
All other cast sources had some instance of quality-assurance procedure performed (ex. UNESCO 1990).
Once all casts were combined by year and formatted, empty, duplicate, and outlier casts were removed (see manuscript).
